# Jokenpô 🪨📄✂️

Um jogo simples de Pedra, Papel e Tesoura desenvolvido com HTML, CSS e JavaScript.


## 📋 Descrição
Este projeto é uma implementação clássica do jogo Jokenpô (também conhecido como Rock Paper Scissors). O jogador compete contra o computador, que faz escolhas aleatórias. O placar é atualizado em tempo real e uma mensagem indica se o jogador ganhou, perdeu ou empatou.


## 🚀 Funcionalidades
- Interface visual intuitiva com botões interativos
- Placar dinâmico que registra pontuações do jogador e do computador
- Feedback visual com mensagens de resultado (ganhou, perdeu ou empatou)
- Design responsivo e agradável


## 🛠️ Tecnologias Utilizadas
- HTML5 para estruturação da página
- CSS3 para estilização e layout
- JavaScript para lógica do jogo e interatividade


## 🎮 Como Jogar
1. Acesse o jogo através do link de deploy
2. Clique em um dos botões: 🪨 (Pedra), 📄 (Papel) ou ✂️ (Tesoura)
3. Veja o resultado da rodada e acompanhe o placar atualizado
